## Why Potions?

- **Low-Maintenance**: Fully automated version detection and releases
- **Reliable**: SHA256 verification and reproducible builds
- **Secure**: Code signing, vulnerability scanning, and SBOM generation
- **Extensible**: Easy addition of new packages via YAML definitions

## Platforms

- **macOS**: Intel (x86_64) and Apple Silicon (ARM64)
- **Linux**: x86_64 and ARM64
